The Performance of Post-Quantum TLS1.3

This page describes the scripts and data necessary to reproduce the evaluation of the paper: "The Performance of Post-Quantum TLS1.3", a collobaration between TUM and Nokia Bell Labs.

The data and scripts can be found in the following places:
  • Scripts including a easy-to-setup docker version of our experiments is available at our main Github Repository.
  • Raw Data from our measurements with hardware timestamping usable for re-evaluation is available on MediaTUM, DOI is 10.14459/2023mp1725057.
  • The used OpenSSL Fork can be found at OpenSSL Github Repository containing a fork of the Open Quantum Safe's project OpenSSL version.
